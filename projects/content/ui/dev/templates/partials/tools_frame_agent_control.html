Как устроены prompt-файлы

prompts/system_prompt.md — основной сценарный prompt для диалога и выбора поведения модели в чате.

prompts/dream_beat_planner.md — Beat Planner (0A), только beats + header_context. Редактор во вкладке Planning → 0A.

prompts/dream_decomposition.md — Сценарист (0B), beats → scenes. Редактор → 0B.

prompts/dream_scene_motion_decompose.md — production decompose_dream_scenes (сцены + motion для старого шага pipeline). Пустой — встроенный дефолт из кода.

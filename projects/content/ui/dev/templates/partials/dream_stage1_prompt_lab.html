Режиссёр

Цепочка: Beat Planner → Сценарист → Режиссёр (одна кнопка «Сформировать план» = два внутренних промпта: референсы по сценаристу, затем кадры с опорой на них) → Сборщик (автоматически получает полный JSON). Ниже, в первую очередь — редактируемые user-контракты (contracts/); системные промпты — в карточках 1A/1B. Справочные схемы JSON — в «Технический уровень» внизу вкладки.

Input от сценариста

То, что уходит в режиссёра: последний output 0B или сохранённый снимок. Кнопка «Сформировать план» сама делает два внутренних шага (референсы по сценаристу → кадры с опорой на референсы). Зафиксируйте input, чтобы новые запуски Сценариста не перезаписали поле.

После запуска Сценариста здесь появится превью header + сцен. Сохранённое состояние подставляется при загрузке.

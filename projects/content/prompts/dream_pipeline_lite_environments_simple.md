Шаг 1 Dream Lite (simple mode): сгенерируй только ## Окружения и ## Персонажи в markdown.

Критично для блока ## Окружения:
- описывай только пространство/локацию как «чистую зону» без живых существ;
- в окружениях запрещены люди, животные, птицы, рыбы, насекомые, толпы, силуэты, персонажи вдалеке и любые биологические существа;
- если в исходном сне есть живые существа, всё равно убирай их из описаний окружения;
- фокус на геометрию, материалы, свет, атмосферу, погоду, цвет, ракурс и зону действия.

Блок ## Персонажи:
- только люди (как отдельные карточки), без взаимодействий и без сцен действия.

Формат:
- markdown с заголовками `## Окружения`, `## Персонажи`;
- карточки `### <id>` и краткие описания;
- без JSON, без местоимения «я», без служебных комментариев.
